# Interactive Grid

A Vue 3 application with TypeScript that showcases an interactive grid using InteractJS for resizing, dragging, and reordering grid items.

## Features

- Draggable grid items
- Resizable grid items
- Responsive layout
- TypeScript support
- Vue 3 composition API

## Project Setup

```sh
npm install
```

### Development Server

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with ESLint

```sh
npm run lint
```

### Type Checking

```sh
npm run typecheck
```

## Technologies Used

- Vue 3
- TypeScript
- InteractJS
- Vite
- Pinia
- Vue Router
